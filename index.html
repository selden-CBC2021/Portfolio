About Me

Greetings! I am Drew Selden. I am an Michigan State University alumni with a Bachelor's degrees in Human Biology and Criminal Justice. I'm currently enrolled in a Coding Bootcamp to learn the basics of computer programming and web development. It's been quite the transition in terms of the way you have to think while coding especially in terms of JavaScript functionality and how they work

Since my enrollment in MSU's Coding Bootcamp I have learned the basics of HTML, CSS, and JavaScript as well as a little JQuery, NodeJS, API calling and functionality and many other practical skills. I'd like to use these new found skills and eventually be able to work in a field of rapidly growing technologies such as blockchain and/or fintech but really anywhere where I can get my feet wet with some real job experience.

Work

The first major milestone has been reached in the bootcamp with completion of my first project(Click on the image above to see) The project was a Covid-19 Risk and Vaccine Assessment application that retrieves real-time Covid-19 cases and Covid-19 vaccine data based on the users geo-location. It includes HTML, CSS, JS and Web-APIs to allow full functionality. I worked with a team of five during this project and the Github practice with branching, merge requests, and the kanban board was a huge benefit to learn how the workflow of a real job would look and feel like.

My latest achievement was the completion of a fully functioning weather app that will display your current location temp, wind, high, low, as well as a few other items and a search bar that will find any city in the U.S. of your choosing. The searches will be saved to local storage and displayed on the screen for quick comparisons.

Another working application I made was a simple work day scheduler that included a current time display using moment.js as well as color codes to indicate if the time slot is in the past present or future and the ability to save each time slot to local storage.
